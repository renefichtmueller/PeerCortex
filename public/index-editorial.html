Integrated Sources of Trust
🌐
PeeringDB
Network profiles, IX presence, facilities, peering policy. The authoritative source for interconnection data.
peeringdb.com
📊
RIPE Stat
Announced prefixes, AS neighbours, routing visibility, BGP updates, geolocation, abuse contacts.
stat.ripe.net
🛡️
RPKI / ROA Validation
Per-prefix Route Origin Authorization validation via RIPE RPKI validators. Detects invalid or missing ROAs.
rpki.cloudflare.com
🔐
ASPA (RFC 9582)
AS Provider Authorization from Cloudflare RPKI JSON feed. RFC-compliant upstream/downstream path verification with valley detection.
IETF Draft-14
📡
RIPE Atlas
Active measurement infrastructure. Probe presence, connectivity status, and anchor detection per ASN.
atlas.ripe.net
🔭
bgproutes.io
Next-gen BGP data collection. 3,294+ vantage points, RIB queries, ROV and ASPA validation status per route.
bgproutes.io
🔍
NLNOG IRR Explorer
Cross-references BGP origin announcements with Internet Routing Registry records. Detects mismatches and unauthorized announcements.
irrexplorer.nlnog.net
🤝
MANRS Observatory
Mutually Agreed Norms for Routing Security. Checks membership, conformance level, and routing security commitment.
observatory.manrs.org
🌍
bgp.he.net
Hurricane Electric BGP Toolkit. AS information, prefix lists, peer counts, and country attribution.
bgp.he.net
🚫
Team Cymru Bogon Reference
Bogon prefix and ASN detection. Identifies reserved, unallocated, and private address space in BGP announcements.
team-cymru.com
📂
RIPE DB / IRR
Internet Routing Registry objects (aut-num, route, as-set). RPSL policy validation and object completeness checks.
apps.db.ripe.net
📈
Route Views / RIPE RIS
BGP route collectors providing global routing table visibility. Used for path analysis, visibility scoring, and anomaly detection.
routeviews.org
All data is queried in real-time from authoritative sources. No data is stored or cached beyond 5 minutes.
